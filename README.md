# Beau Teeth Ful - Website

A professional static website for Beau Teeth Ful, a cosmetic teeth whitening and tooth gems service based in Manurewa, Auckland, New Zealand.

## Services Offered

- **Teeth Whitening** - Professional whitening treatments for a brighter smile
- **Tooth Gems** - 18k gold tooth gems with drill-free procedure

## Features

- Responsive design (mobile, tablet, and desktop)
- Modern and elegant UI with golden/luxury theme
- Smooth animations and transitions
- Image gallery showcasing services
- Contact form for appointment booking
- Social media integration (Facebook, Instagram, TikTok)
- SEO-friendly structure

## How to Use

1. Simply open `index.html` in any modern web browser
2. The website is fully static and doesn't require a server
3. All mock images are SVG-based placeholders that can be replaced with actual photos

## Customization

### Replace Mock Images
The current images are SVG placeholders. To use real photos:
1. Take high-quality photos of teeth whitening results and tooth gems
2. Save them in the `images/` folder with the same filenames
3. The website will automatically display your photos

### Update Contact Form
The contact form currently shows an alert. To make it functional:
- Integrate with [FormSpree](https://formspree.io/)
- Use [EmailJS](https://www.emailjs.com/)
- Set up [Netlify Forms](https://www.netlify.com/products/forms/)
- Connect to your own backend server

### Colors and Branding
The color scheme can be customized in `styles.css`:
```css
:root {
    --primary-color: #e6b980;
    --secondary-color: #f4d9b8;
    --accent-color: #c89f66;
}
```
